# AI-Powered Job Portal

An AI-powered job portal that helps job seekers improve their resumes, discover matching opportunities, and allows recruiters to manage hiring efficiently.

## Tech Stack

- React
- Node.js
- Express
- MongoDB
- OpenAI API
- Tailwind CSS

## Features

- JWT Authentication
- Resume Upload
- AI Resume Analysis
- AI Job Recommendations
- Recruiter Dashboard
- Email Notifications
